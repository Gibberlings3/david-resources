Copy Portraits and Sounds

Author: DavidW

Version: beta 1
Languages: English
Platforms: Windows

Overview

This is a simple mod that just copies the portraits and sounds between different Enhanced Edition games (specifically, between the enhanced versions of Icewind Dale, Baldur's Gate, and Baldur's Gate II). All relevant text strings (for instance, sound subtitles) are also copied. The portraits and soundsets are inserted into the game in the same way as original-game resources, hopefully seamlessly. (For instance, portraits will be separated by sex, class and race.) The mod also installs the portraits from Icewind Dale II, and has components to remove the portraits of joinable BG and BG2 characters and to set the default colors of characters to match their portraits. You will (of course) need to have the various games installed to do a resource copy.

(Why copy resources over rather than just shipping them with the mod? Partly download time (I accept this is not a big deal these days). Partly future-proofing against subsequent game updates. Mostly because I feel slightly uncomfortable just bulk copying resources like this between games, especially actively-supported games. I accept this is a gray area.)

Finding your Games

Because this mod copies resources between different Infinity Engine games, it has to be able to find those games. By default, it assumes that any games you have are sitting in the same directory as your current game folder. (So if, for instance, you have Baldur's Gate: Enhanced edition in the folder 'C:\games\bgee', the mod assumes that Icewind Dale:Enhanced Edition is also in some folder in C:\games, such as C:\games\iwdee. (If the folder is there, it doesn't matter what it's called.)

This should cover most standard installation patterns. However, if you have a more exotic setup you can tell the mod where to look explicitly. Open dw_port_snd/dw_port_snd.ini, and edit game_location_iwdee (and similar) to give the exact path of your game. (e.g. C:\games\bgee).

If you have multiple copies of the games installed, the mod defaults to the most recent copy (based on the date in build-date.txt, and only to the nearest year.)

Compatibility

Copy Portraits and Sounds is compatible with any of Baldur's Gate: Enhanced Edition (with or without Siege of Dragonspear), Baldur's Gate II: Enhanced Edition, and Icewind Dale: Enhanced Edition. It is probably compatible with Enhanced Edition Trilogy, but I haven't tested it. It is not compatible with pre-Enhanced-Edition versions of the games.

Copy Portraits and Sounds makes some tweaks to the game's user interface, so it should be installed after any mod that does a wholesale revision of the user interface. The mod is fairly light-touch and compatibility-friendly in its own edits, so I don't anticipate incompatibility with other mods that edit the user interface... but UI modding is a complicated profession so it is always possible something might creep through.

Copy Portraits and Sounds only works on Windows. It would probably be pretty trivial to modify it to work on OSX or Linux, but I lack the knowledge and the testing environment.

Installation

(This is not currently correct as I'm not bundling a release, but it would be if I released it.)

Copy Portraits and Sounds is distributed as a self-extracting archive and includes a WeiDU installer. To install, simply double-click the archive and follow the instructions on screen.

Alternatively, the files can be extracted into your game directory using 7zip or WinRAR. When properly extracted, your game directory will contain setup-dw_port_snd.exe and the folder dw_port_snd. To install, double-click setup-dw_port_snd.exe and follow the instructions on screen.

You can run setup-dw_port_snd.exe in your game folder to reinstall, uninstall or otherwise change components.

Note for Complete Uninstallation
In addition to the methods above for removing individual components, you can completely uninstall the mod using setup-dw_port_snd --uninstall at the command line to remove all components without wading through prompts.

Contents

Include portraits from Baldur's Gate II: Enhanced Edition (BGEE, IWDEE)

This component collects and installs the portraits from BG2EE. You must have a copy of BG2EE installed, and the mod must be able to find it. (See above.)

Include portraits from Icewind Dale: Enhanced Edition (BGEE, BG2EE)

This component collects and installs the portraits from IWDEE. You must have a copy of IWDEE installed, and the mod must be able to find it. (See above.)

Include portraits from Icewind Dale II (IWDEE, BGEE, BG2EE)

This component collects and installs the portraits from IWD II. (It's not straightforward to pick these up, so the mod contains the files directly.)

Hide portraits used by joinable NPCs (BGEE, BG2EE)

Many of the portraits in BGEE and BG2EE are shared by joinable non-player characters. This component just hides those portraits in character creation so that you can be confident of not picking one.

Match default colors to portraits (IWDEE, BGEE, BG2EE)

Borrowed from Icewind Dale II, this is a convenience/immersion tweak that presets the colors of your character in character generation to match your portrait. (You can change them freely; it's just the starting values that are set.) This works for any of the portraits from BGEE, BG2EE, IWDEE, and IWD2.

Collect soundsets from Icewind Dale: Enhanced Edition (BGEE, BG2EE)

This component collects and installs the soundsets from IWDEE. You must have a copy of IWDEE installed, and the mod must be able to find it. (See above.)

Collect soundsets from Baldur's Gate: Enhanced Edition (IWDEE, BG2EE)

This component collects and installs the soundsets from BGEE (including the ones in Siege of Dragonspear if you have it installed). You must have a copy of BGEE installed, and the mod must be able to find it. (See above.)

Collect soundsets from Baldur's Gate II: Enhanced Edition (IWDEE, BGEE)

This component collects and installs the soundsets from BG2EE. You must have a copy of BG2EE installed, and the mod must be able to find it. (See above.)
